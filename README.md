<h2>功能</h2>
1. 支持模拟定位以及模拟导航<br>
2. 历史记录存储<br>
3. 选址模块<br>
4. 支持Android13，其他版本手机目前手机没有真机，手里只有一加8T<br>
5. 支持文件数据读取&文件数据生成工具<br>
6. 通过广播形式启动模拟导航<br>
